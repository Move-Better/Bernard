Fixes the missing Words tab

editorArchetype.js's doc archetype rails on a 'doc' key that UnifiedEditor's RAIL_META never defines — it silently drops. Renaming it to 'words' (the same key carousel/visual/ad/textad already use) is the whole fix; the existing panel-routing code needs no new branch.

Regenerate & style switch

Ported verbatim from the pre-#2107 AssetsPane.jsx: same confirm-before-regenerate chip, same length-preset picker, same useRegenerateBlogStreamed mutation (Opus via /api/stream, no function-cap timeout).

Split into series

Same 2/3/4-parts confirm dialog and useSplitBlogIntoSeries mutation. On success, navigates to the new Part 1 (?piece= in the old console → the editor's own piece param).

Scoped to blog only

Style switch and split are already blog-gated in the ported code. Regenerate is wired to useRegenerateBlogStreamed only here — SlideEditor's CaptionPanel and StoryComposer have no regenerate today either (confirmed), but re-adding it there is a separate follow-up, not bundled into this fix.
