Highlight-to-style — full Canva-style toolbar

Double-click into the caption and select any word or phrase. The toolbar carries the Canva set: font, size, color, B I U S, case, align, spacing, and effects. Green-tagged controls restyle just the selection; grey-tagged ones apply to the whole text box.

Select a word to style it

Toolbar controls

  • AaFont per-word — swap the typeface on a selection.
  • −#+Size per-word — numeric, like Canva. Emphasise one word without a 2nd box.
  • AColor per-word — brand swatches + custom.
  • BIUSBold / Italic / Underline / Strikethrough per-word.
  • aACase per-word — UPPER / lower / Sentence.
  • ≡Align whole box — left / center / right.
  • ↕Spacing whole box — letter + line.
  • ✦Effects whole box — shadow depth for legibility on photos.
Dropped from Canva's bar (ask if you want them): Lists (not used in captions), Animate (still image), Position (you already drag on canvas).

Color ships today; everything else is new work in the publish compositor so what you see here bakes into the real post.
