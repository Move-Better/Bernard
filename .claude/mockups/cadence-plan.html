What this is

The cadence is a per-workspace policy object (targets, digest frequency, feeds, quiet days) Bernard recommends & self-tunes, you override in plain language, and every producer/scheduler reads. Default is smart so a non-expert is served by reacting; knobs are always there but never required. Bernard produces to fill these targets and holds surplus as backlog — never floods.
